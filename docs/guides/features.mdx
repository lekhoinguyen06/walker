These are the common Walker features we aim to provide across all our libraries. This page describes their purposes. To learn how to use them, refer to the framework and language documentation.

## Behaviours

These are the default Flows provided:

- Click
- Scroll
- Capture
- Continue/Stop/Cancel
- Navigate
- Confirm
- Decide
- Fill input

## UI

Community UI packages from Concierge and our contributors help you quickly integrate Walker into your application. We provide two package types: styled and unstyled.
