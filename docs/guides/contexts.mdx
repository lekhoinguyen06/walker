title: "Contexts"

---



## Front 

Embedded in an HTML element as JSON in the `walker-data` attribute, you can provide component-level context on how to use your website in Items.

You should declare Front context for necessary guidelines and descriptions. Users will benefit if they use a personal agent that run on their browser, such as OpenClaw. If these personal agents adhere to these guidelines, they will consume your website appropriately.

## Back

For every Walker component on your website, if you use Concierge or your own Walker Server, you can declare additional backend-level context.

You should declare Back context for privilege information, such as the latest promotions or product descriptions. Concierge helps you manage the agent's Back context, try it out!

## Hook

The Walker library allows you to use hooks to provide custom context for your agent.

When you need customization, you can declare hooks. For example, you can declare a database hook allows your agent to have transactional awareness. Concierge provides a platform to observe, authenticate, or governance your hooks.
